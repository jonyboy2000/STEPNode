Overview

The STEP-NC DLL can be used to develop custom advanced control solutions and other applications. For a general overview of the libraries design and structure, you may want to start with this White Paper [pdf]

The DLL is part of the STEP-NC Machine package and you can use it to build your own local STEP-NC applications on any machine that the STEP-NC Machine is installed on. Contact STEP Tools if for more information about the STEP-NC Machine SDK that includes a license for distributing applications based in the STEP-NC DLL.
