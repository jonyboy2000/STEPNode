Overview

The STEP-NC API is used by the NC.js STEP-NC Node interface and the STEP-NC .NET interface, as well as a lower level C++ library. It can be used to develop custom advanced control solutions and other applications. Please refer to each of those packages for specifics on setup and environment.

The API has several different categories of functions as shown below:

  • Introduction
  • APT Object
  • Finder Object
  • Tolerance Object
  • Feature Object
  • Process Object
  • Machine State
